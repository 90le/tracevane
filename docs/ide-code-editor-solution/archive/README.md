# 历史执行记录归档

本目录保存已经完成阶段的执行计划、进度日志、验证证据和决策记录。

这些文件不是当前方案主线入口；当前设计和后续阶段以 `../00-README.md`、`../01-产品边界与形态拆分.md` 至 `../08-实施阶段验收与风险.md`、以及 `../10`–`../13` 为准。

归档文件用途：

| 文件 | 用途 |
|---|---|
| `m1-execution-plan.md` | M1 File Manager Online Editor 执行计划与完成口径 |
| `m1-progress.md` | M1/M1.x 进度、验证证据、风险记录 |
| `m1x-execution-plan.md` | M1.x 编辑器窗口、Tab、安全保存、状态栏、入口增强执行计划 |
| `m2-execution-plan.md` | M2 Unified File Surface 与 Monaco 能力补齐执行计划 |
| `m2-progress.md` | M2/M2.x 进度、验证证据、风险和决策日志 |

维护规则：

- 不把新阶段计划继续追加到归档文件。
- 如需追溯 M1/M2 为什么这样实现，优先查看对应 progress 文件。
- 如归档文件中的旧路径与当前主线冲突，以当前主线文档为准。
