# 终端能力设计

> 状态：Active design
> 更新：2026-06-25

## 1. 定位

终端是 Workspace 的本地运行工具，不是 CLI Agent。

它负责 shell、命令、task、dev server、logs、ports 和 terminal evidence。

它不负责 Codex/Claude/OpenCode runtime readiness、Agent Run 生命周期或模型路由配置。

## 2. 与文件管理的硬边界

终端不能替代文件管理：

- 终端可以从文件管理接收 cwd 或路径参数。
- 终端可以把输出、命令结果、日志片段加入 Evidence 或 Agent context。
- 终端不能拥有上传、下载、批量删除、重命名、移动、压缩解压等文件管理 UI。
- 终端错误、stale session、reconnect 状态不能污染文件管理页面。
- 文件管理的安全确认和批量影响范围必须由 Files/File Manager 呈现，而不是要求用户读 shell 输出。

## 3. 目标

终端能力必须比当前版本更稳：

- session 可恢复；
- 输出可 replay；
- 断线可重连；
- stale session 自动清理；
- 移动端可实际输入；
- task/ports/logs 一等化；
- 输出可加入证据或 Agent 上下文。

## 4. 前端能力

PC：

- session tabs；
- rename；
- new/duplicate；
- close/kill；
- reconnect；
- clear local view；
- search output；
- copy selected output；
- fullscreen；
- send output to evidence/context；
- task launcher；
- ports inspector。

Mobile：

- full-screen terminal；
- input accessory bar；
- paste command；
- Ctrl+C；
- Esc/Ctrl/Tab/Arrows；
- reconnect/clear/copy/search in action sheet；
- large touch targets。

## 5. 后端能力

必须有：

- registry；
- pty lifecycle；
- output ledger；
- SSE/WebSocket reconnect；
- replay from seq；
- heartbeat；
- stale cleanup；
- bounded buffer；
- resize debounce；
- structured errors。

## 6. 错误体验

禁止首屏直接显示裸错误如：

```text
[error: terminal_session_not_found]
```

应该显示：

- 会话已失效；
- 可“新建终端”或“从列表移除”；
- 若可恢复，提供“重连”；
- 错误详情折叠到 evidence/log。

## 7. 与 Agent 的关系

允许：

- terminal output 作为 Agent context；
- terminal command result 作为 verification evidence；
- Agent 建议命令可在 Terminal 中经确认执行。

禁止：

- terminal session = CLI Agent Run；
- CLI Agents 管理 terminal tabs；
- terminal 页面显示 Provider/模型配置。

## 2026-06-26 Workspace 终端会话创建与不可恢复会话治理

Workspace 终端面板不再通过直接打开 SSE stream 来隐式创建终端。HTTP stream attach 仍保持严格语义：只能 attach 已存在的运行时 PTY，会话不存在时返回错误，避免误把历史 session id 自动复活。前端新建终端先调用 `POST /api/terminal/sessions` 创建一个 detached PTY descriptor，再用 `/api/terminal/sessions/:id/stream` attach 到同一个会话。

设计约束：
- `terminal_session_not_found` 不应作为正常 UI 输出出现在终端黑屏里。
- 已完成、失败或 lost 的历史会话是 evidence，不是可 attach 的 live terminal；Workspace roster 展示但禁用选择。
- 新建、attach、input、resize 属于终端会话生命周期；文件管理器只可 handoff cwd，不接管终端创建。
