02 / Install

Inspect first.
Then install.

The release installer verifies metadata and package SHA-256. Tracevane deliberately avoids the opaque curl | bash pattern.

1
Download and review

Save the installer locally and inspect it.

2
Check the release

Verify the version, source, and checksum.

3
Choose a mode

Standalone service or existing Gateway.

INSTALL · BASH
# Requires an installed and initialized OpenClaw
# Download the latest signed release installer
$ curl -fL https://github.com/90le/tracevane/releases/latest/download/install-tracevane.sh -o /tmp/install-tracevane.sh
$ sed -n '1,220p' /tmp/install-tracevane.sh
$ chmod +x /tmp/install-tracevane.sh

# Verify, then install in standalone mode
$ /tmp/install-tracevane.sh --check-release
$ /tmp/install-tracevane.sh --mode standalone --json

Gateway mode: replace --mode standalone with --mode gateway. Linux, macOS, and WSL are supported.
