OpenClaw Studio · 小丘

OpenClaw Studio 是一个业余项目，是 OpenClaw 的 Web 管理控制台。把命令行的全部能力搬进浏览器——实时对话、Agent 管理、频道配置，一个页面全搞定。

以原生插件运行，支持独立端口或挂载到 Gateway。Vue 3 前端 + 模块化后端，流式输出、历史恢复、工具调用可视化开箱即用。

安装指南

现在统一提供网站一键安装脚本。为了避免 AI/安全策略拦截，这里不再使用在线脚本直通执行，而是先下载到本地、可选检查内容，再本地运行。请选择非单口或单口模式后再安装。

👤 模式一：非单口

本地访问，或可以直接开放 3760 端口的机器
🏢 模式二：单口

外网只开放 OpenClaw 端口，Studio 挂载到 /studio，同时保留本机 3760

模式一：独立端口 STANDALONE

请在我的 OpenClaw 环境里安装 OpenClaw Studio，使用非单口模式（standalone，独立端口 3760）。

请按下面步骤执行，不要使用在线脚本直通执行：

1. 先下载 https://studio.90le.cn/install-openclaw-studio.sh 到本地当前目录
2. 可选执行：sed -n '1,160p' ./install-openclaw-studio.sh
3. 执行：chmod +x ./install-openclaw-studio.sh
4. 执行：./install-openclaw-studio.sh --mode standalone

脚本会自动完成这些事情：
1. 检查并升级 OpenClaw 到最低要求 {{OPENCLAW_MIN_VERSION}}
2. 下载并解压 openclaw-studio-{{STUDIO_VERSION}}.tar.gz
3. 自动修正发布包入口元数据，确保宿主能正确加载 studio 插件
4. 安装依赖并重建 node-pty
5. 写入 plugins.entries.studio / plugins.load.paths / transport 配置
6. 执行 openclaw gateway install --force 与重启
7. 对 http://127.0.0.1:3760/api/system/health 做健康检查

执行完成后，请把以下结果告诉我：
- OpenClaw 版本
- 配置文件路径
- 安装目录
- 访问地址
- 健康检查结果

模式二：单口模式 GATEWAY /STUDIO

请在我的 OpenClaw 环境里安装 OpenClaw Studio，使用单口模式（gateway，共用 OpenClaw 端口，挂载到 /studio，同时保留本机 3760 standalone 入口）。

请按下面步骤执行，不要使用在线脚本直通执行：

1. 先下载 https://studio.90le.cn/install-openclaw-studio.sh 到本地当前目录
2. 可选执行：sed -n '1,160p' ./install-openclaw-studio.sh
3. 执行：chmod +x ./install-openclaw-studio.sh
4. 执行：./install-openclaw-studio.sh --mode gateway

脚本会自动完成这些事情：
1. 检查并升级 OpenClaw 到最低要求 {{OPENCLAW_MIN_VERSION}}
2. 下载并解压 openclaw-studio-{{STUDIO_VERSION}}.tar.gz
3. 自动修正发布包入口元数据，确保宿主能正确加载 studio 插件
4. 安装依赖并重建 node-pty
5. 写入 plugins.entries.studio / plugins.load.paths / gateway transport 配置，并保留 3760 standalone 监听
6. 补齐 gateway.bind / gateway.auth.token / gateway.controlUi.allowedOrigins 等最低可用配置
7. 执行 openclaw gateway install --force 与重启
8. 对 http://127.0.0.1:<gateway_port>/studio/api/system/health 和 http://127.0.0.1:3760/api/system/health 做健康检查

执行完成后，请把以下结果告诉我：
- OpenClaw 版本
- 配置文件路径
- 安装目录
- Gateway 端口
- 3760 回退入口健康检查
- 访问地址
- 健康检查结果
